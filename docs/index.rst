odyCy
=====

odyCy is a :ref:`state of the art <performance>` NLP library for Ancient Greek, capable of
part-of-speech tagging, morphological analysis, dependency parsing, lemmatization and more.

It is based on the rightfully popular and widely used `spaCy <https://spacy.io/>`_ framework, which makes odyCy easy to use, scalable,
reliable and modular.
